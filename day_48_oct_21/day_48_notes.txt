JSON Web Token
==============
-JSON Web Tokens (JWT) are tokens generated by the server upon user authentication on a web application, and then sent to the client.
-These tokens are then sent on every HTTP request, which allows the server to authenticate the user.
-A JSON Web Token consists of: 1.Header  2.Payload  3.signature
